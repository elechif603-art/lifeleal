# LifeLeal - Postal Code & Regional Data Blog

A comprehensive Jekyll-based blog dedicated to postal codes, regional data, and geographic information.

## Features

- **50+ Articles** on postal codes and regional data
- **Google AdSense Ready** - Optimized for monetization
- **Responsive Design** - Works on all devices
- **SEO Optimized** - Built-in sitemap, robots.txt, and meta tags
- **Clean, Modern Layout** - Professional appearance
- **Rich Content** - Detailed guides with tables and data visualization

## Installation

1. Clone this repository
2. Install dependencies: `bundle install`
3. Build and serve: `bundle exec jekyll serve`
4. Visit `http://localhost:4000/lifeleal` in your browser

## Configuration

1. Update `_config.yml` with your GitHub username and site URL
2. Replace Google AdSense and Analytics IDs with your own
3. Update contact form endpoint in `contact.md`

## Deployment

This site is ready to be deployed on GitHub Pages:

1. Push to your GitHub repository
2. Enable GitHub Pages in repository settings
3. Your site will be available at `https://www.lifeleal.com.ng`

## Content

All articles are original, human-written content covering:

- Regional demographics and data
- Geographic information guides
